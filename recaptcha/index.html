reCAPTCHA 테스트 페이지

테스트 페이지 목록

  • reCAPTCHA v2 테스트
  • reCAPTCHA v3 테스트
  • 서버 상태 확인

사용 방법

  1. Google reCAPTCHA 콘솔에서 Site Key와 Secret Key를 발급받으세요
  2. 도메인 설정에 localhost:3000을 추가하세요
  3. Spring Boot 서버를 8080 포트에서 실행하세요
  4. 각 테스트 페이지에서 Site Key를 입력하고 테스트하세요

설정 정보

  • 웹 서버: http://localhost:3000 (Python 서버)
  • API 서버: http://localhost:8080 (Spring Boot)
  • API 엔드포인트:
    • GET /open/recaptcha/status
    • POST /open/recaptcha/v2/verify
    • POST /open/recaptcha/v3/verify

문제 해결

  • CORS 오류: Spring Boot에서 CORS 설정 확인
  • 연결 오류: API 서버가 8080 포트에서 실행 중인지 확인
  • reCAPTCHA 로드 실패: Site Key가 올바른지 확인
  • 도메인 오류: Google reCAPTCHA 콘솔에서 localhost:3000 도메인 등록 확인
